# Customer Segmentation and Retention Analysis using RFM and Cohort Analysis

## Project Overview

This project focuses on analyzing customer purchasing behavior using **RFM (Recency, Frequency, Monetary) Analysis** and **Cohort Analysis**.

The objective is to segment customers based on their purchasing patterns, identify high-value customers, understand customer retention trends, and provide actionable business insights.

This analysis helps businesses improve customer engagement, create targeted marketing strategies, and increase customer lifetime value.

---

## Business Problem

Businesses have large amounts of transaction data but need a way to identify:

* Who are the most valuable customers?
* Which customers are becoming inactive?
* How frequently do customers purchase?
* How much revenue does each customer generate?
* How well are new customers being retained over time?

This project solves these problems using customer segmentation and retention analysis.

# RFM Analysis

## What is RFM?

RFM analysis is a customer segmentation technique based on three customer behaviors:

| Metric    | Meaning                             |
| --------- | ----------------------------------- |
| Recency   | How recently the customer purchased |
| Frequency | How often the customer purchases    |
| Monetary  | How much money the customer spends  |

---

## RFM Calculation

### Recency

Calculated as the number of days since the customer's latest purchase.

```
Recency = Analysis Date - Last Purchase Date
```

### Frequency

Number of unique transactions made by each customer.

```
Frequency = Count of Purchases
```

### Monetary

Total revenue generated by each customer.

```
Monetary = Sum of Purchase Amount
```

# Cohort Analysis

## What is Cohort Analysis?

Cohort Analysis groups customers based on their first purchase month and tracks their purchasing behavior over time.

It helps measure:

* Customer retention
* Customer loyalty
* Long-term engagement

---

## Cohort Calculation Steps

1. Identify customer's first purchase month.

2. Assign customers into monthly cohorts.

3. Calculate the number of active customers in each month after acquisition.

4. Generate retention percentage.
